# Canadian Employment Trends Analysis (2019–2025)

## Overview

This project explores employment trends across major Canadian industries from 2019 to 2025 using data from Statistics Canada. The analysis focuses on understanding the impact of the COVID-19 pandemic on employment levels and examining how different industries recovered during the post-pandemic period.

Using Python, Pandas, Matplotlib, and Seaborn, the project performs exploratory data analysis (EDA) to identify employment patterns, industry growth rates, employment volatility, and sector-specific recovery trends.

## Key Topics

* Pre-COVID employment structure in Canada
* Employment losses during the initial COVID-19 shock (2020)
* Industry recovery and growth from 2020 to 2025
* Employment volatility across industries
* Long-term labour market trends

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Source

Statistics Canada – Labour Force Survey Employment Data
